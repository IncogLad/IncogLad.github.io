Game of the Generals

SFML | Board Game | Strategy | AI | PC

Project Description

A classic strategy board game from the Philippines wherein 2 players are pit against each other managing there pieces to win the game. In this project, the player gets to play with a programmed AI.

This game is an academic project developed for the GDINTAI (Introduction to AI Algorithms) course. This is a solo project where I programmed all the gameplay features needed, such as the enemy AI algorithm, gameplay logic, pieces hierarchy, object transform, and piece status. The assets were created by Robert Ryan Cabatuando.

Japanese House

Unity | Unity-Probuilder | 3D Space Showcase

Project Description

A japanese house 3D space built and designed in Unity. This project showcases the some of the 3D graphical features available in Unity such as dynamic materials, lighting, particle effects, and post processing effects.

This 3D space showcase is an academic project developed for the GDFUNDA (Game Development Fundamentals) course. I grouped up with 3 other students to design and build this 3D space.

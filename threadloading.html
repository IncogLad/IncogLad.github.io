I'm Okay Loading Screen

SFML | Candle-SFML | Loading Screen | Multithreaded | PC

Project Description

An SFML program simulating a multithread interactive loading screen of a game, which is inspired from I'm Okay, our game entry for the Global Game Jam 2023.

This program is an academic project developed for an elective course about parallel computing and multithreading. This is a solo project such that I designed the interactive loading screen and its multithreaded asset loading. The assets were created by our game artist, Angelic Aram, during the Global Gam Jam 2023.
